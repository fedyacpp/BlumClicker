# BlumClicker

**Версия:** 2.1.0 β
**Последнее обновление:** 19 марта 2025 г.

Я советую удалить `settings.json`, чтобы избежать ЛЮБЫХ возможных конфликтов; теперь вы можете выбрать язык ui и окно для захвата, проверьте окно настроек! Это было очень трудное для написания обновление из-за обстановки в моей личной жизни, поэтому, если вы обнаружите какие-либо баги, пожалуйста, сообщите о них.

BlumClicker — это инструмент автоматизации для игры Drop Game, работающий на основе технологии компьютерного зрения YOLOv11 для точного обнаружения и взаимодействия со снежинками. Инструмент имеет графический интерфейс, широкие возможности настройки и интеллектуальную автоматизацию игрового процесса.

## Возможности

- **Распознавание окон** — Автоматически обнаруживает и фокусируется на окне игры Telegram
- **Обнаружение объектов YOLOv11** — Современное визуальное распознавание целей-снежинок
- **Технология OCR** — Обнаруживает и нажимает кнопку "Играть" для автоматического перезапуска игры
- **Статистика в реальном времени** — Мониторинг показателей производительности, количества кликов и использования системных ресурсов
- **Графический интерфейс** — Визуализация активности бота с панелями и обновлениями в реальном времени
- **Настраиваемые параметры** — Настройка задержек, ограничений FPS, порогов обнаружения и многое другое
- **Визуализация отладки** — Опциональная визуальная обратная связь, показывающая границы обнаружения и оценки достоверности
- **Управление горячими клавишами** — Удобные комбинации клавиш для всех основных функций

## Системные требования

### Минимальные рекомендуемые требования
- **Операционная система:** Windows 10 (64-бит)
- **Процессор:** Intel Core i5 или эквивалент AMD Ryzen 5
- **ОЗУ:** 8 ГБ
- **Видеокарта:** NVIDIA GPU с поддержкой CUDA (протестировано с CUDA 11.8 и 12.5)
- **Python:** Версия 3.12.x
- **Хранилище:** 2 ГБ свободного места
- **Программное обеспечение:** Настольное приложение Telegram

### Рекомендуемая конфигурация
- **Операционная система:** Windows 10/11 (64-бит)
- **Процессор:** Intel Core i7/i9+ или AMD Ryzen 7/9+
- **ОЗУ:** 16 ГБ+
- **Видеокарта:** NVIDIA GPU с поддержкой CUDA (протестировано с CUDA 11.8 и 12.5)
- **Python:** Версия 3.12.x
- **Хранилище:** 5 ГБ+ свободного места

## Установка

1. **Клонирование репозитория**
   ```bash
   git clone https://github.com/fedyacpp/BlumClicker.git
   cd BlumClicker
   ```

2. **Установка необходимых пакетов Python**
   
   Для пользователей GPU (рекомендуется для лучшей производительности):
   ```bash
   pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
   pip install -r requirements.txt
   ```
   
   Для пользователей CPU:
   ```bash
   pip install torch torchvision torchaudio
   pip install -r requirements.txt
   ```

3. **Загрузка файла модели**
   
   Загрузите предварительно обученный файл модели YOLO:
   - [best.pt](https://drive.google.com/file/d/1lUTl4GulseoWs_vhPnYp0qkIYaumKMNg/view?usp=sharing)
   - Поместите его в корень папки со скриптом

4. **Проверка доступности CUDA** (только для пользователей GPU)
   ```bash
   python isCudaAvailable.py
   ```

5. **Запуск BlumClicker**
   ```bash
   python main.py
   ```

## Использование

1. **Запуск приложения**
   ```bash
   python main.py
   ```

2. **Обнаружение окна**
   - Бот автоматически обнаружит ваше окно Telegram

3. **Управление**
   - Используйте горячие клавиши для управления ботом (см. раздел [Горячие клавиши](#горячие-клавиши))
   - Основной интерфейс отображает статистику в реальном времени и информацию о состоянии
   - Доступ к настройкам через CTRL+W для настройки поведения

4. **Режимы работы**
   - Режим автоматической игры: Автоматически перезапускает игру, когда она заканчивается
   - Ручной режим: Требует ручного перезапуска игр
   - Режим отладки: Показывает визуальную обратную связь обнаружения

## Настройка

BlumClicker предлагает возможности настройки через панель настроек (CTRL+W):

### Настройки времени
- **Задержка между кликами** — Время ожидания между последовательными кликами (секунды)
- **Задержка перед кликом** — Время ожидания перед выполнением клика (секунды)
- **Ограничение FPS** — Максимальное количество обрабатываемых кадров в секунду
- **Количество повторных попыток** — Количество повторных попыток для операций

### Параметры функций
- **Автоматическая игра** — Автоматически начинать новые игры
- **Окно отладки** — Показывать визуальную обратную связь обнаружения
- **Кликать по всем бомбам** — Кликать по всем обнаруженным объектам, а не только по целям
- **Режим CPU** — Принудительная обработка только на CPU (отключение GPU)
- **Звуковые эффекты** — Включение/отключение звуковой обратной связи для кликов

### Настройки модели
- **Путь к модели** — Расположение файла модели YOLO
- **Перезагрузка модели** — Возможность горячей замены моделей во время работы

## Горячие клавиши

| Комбинация | Функция |

| CTRL+Q | Выход из приложения |

| CTRL+R | Перезагрузить модель |

## Устранение неполадок

### Распространенные проблемы

| Проблема | Решение |
|----------|---------|
| **Окно не обнаружено** | Убедитесь, что Telegram запущен и виден; попробуйте CTRL+F для повторного обнаружения |
| **Ошибки CUDA** | Проверьте установку GPU и CUDA; при необходимости переключитесь в режим CPU |
| **OCR не работает** | Убедитесь, что зависимости EasyOCR установлены правильно |
| **Неточное обнаружение** | Убедитесь, что файл модели правильно загружен и работает на поддерживаемом GPU |
| **Ошибки разрешений** | Запустите от имени администратора для правильного доступа к окну |

### Логи и диагностика

- Проверьте `bot_log.txt` для получения подробной информации об ошибках
- Включите режим отладки (CTRL+D) для визуализации производительности обнаружения
- Используйте панель настроек для просмотра использования системных ресурсов и последних ошибок

## Часто задаваемые вопросы

**В: Не нарушает ли это условия использования Telegram?**  
О: Инструменты автоматизации могут нарушать условия использования Telegram. Используйте на свой страх и риск.

**В: Могу ли я использовать несколько экземпляров для разных игр?**  
О: Технически да, но скрипт не будет работать как задумывался сразу из коробки.

**В: Работает ли BlumClicker на MacOS/Linux?**  
О: В настоящее время он работает только на Windows из-за специфических механизмов обработки окон.

**В: Насколько точно обнаружение?**  
О: С YOLOv11 и правильной настройкой точность обнаружения обычно превышает 95%.

**В: Могут ли заблокировать мой аккаунт?**  
О: Используйте на свой страх и риск. Инструмент пытается имитировать поведение человека, но гарантий нет.
